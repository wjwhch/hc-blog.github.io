    # hasOwnProperty

    hasOwnProperty: 是用来判断一个对象是否有你给出名称的属性或对象。不过需要注意的是，此方法无法检查该对象的原型链中是否具有该属性，该属性必须是对象本身的一个成员。

    ECMAScript 中的所有对象都继承自Object，

    Object拥有如下属性：

    1. constructor: 对创建对象的 函数 的引用（指针）。对于 Object 对象，该指针指向原始的 Object() 函数。
    2. Prototype: 对该对象的对象原型的引用。对于所有的对象，它默认返回 Object 对象的一个实例。

    Object拥有如下方法：

    1. hasOwnProperty( property) : 判断对象是否有某个特定的属性。必须用字符串指定该属性。 （例如，o.hasOwnProperty("name")）
    2. IsPrototypeOf(object): 判断该对象是否为另一个对象的原型。
    3. PropertyIsEnumerable:: 判断给定的属性是否可以用 for...in 语句进行枚举。
    4. ToString(): 返回对象的原始字符串表示。对于 Object 对象，ECMA-262 没有定义这个值，所以不同的 ECMAScript 实现具有不同的值。
    5. ValueOf(): 返回最适合该对象的原始值。对于许多对象，该方法返回的值都与 ToString() 的返回值相同。

    # for in 和 for of

    for in: 便利对象和数组的 可枚举 属性，返回对象的key，数组的下标；

    • **遍历数组时的缺点：**迭代顺序依赖执行环境，所以数组遍历不一定按次序访问元素。因此当迭代访问数序很重要的数组时，最好用for循环或者Array.prototype.forEach()或者for of;

    for fo: 可以遍历***set map arrary string***; 由于对象没有迭代器，所以无法遍历。

    # arguments 和 剩余参数(Rest Parameters)

    arguments: 包含函数所有实参的类数组对象；

    • es6的箭头函数没有this和arguments。

    剩余参数(Rest Parameters): function(one, ...two){} // tow就是剩余所有参数，是数组。

    1. 剩余参数必须是最后的参数，不然会报错
    2. 不能用在setter方法中中，因为setter方法只接收单个参数。

    # 订阅-发布

    class eventEmit {
  constructor() {
    // 用来存储事件和监听函数之间的关系
    this.eventMap = {}
  }

  // type: 事件名称； handler: 回调函数
  on(type, handler) {
    // 判断是否传入函数
    if (!(handler instanceof Function)) {
      throw new Error('请传一个函数');
    }
    // 判断是否有该队列存在
    if (!this.eventMap[type]) {
      // 不存在就新建队列
      this.eventMap[type] = []
    }
    // 存在就往队列里推入handler
    this.eventMap[type].push(handler);
  }

  // type: 事件名称； params: 回调参数
  emit(type, params) {
    // 判断是否有该队列
    if (this.eventMap[type]) {
      // 存在，就依次执行该队列的handler
      this.eventMap[type].forEach(handler => {
        handler(params)
      })
    } else {
      // 不存在，报错
      throw new Error("找不到对应事件")
    }
  }

  // 卸载订阅
  off(type, handler) {
    // 判断是否存在该队列
    if (this.eventMap[type]) {
      // 存在则移除该回调在队列内移除
      this.eventMap[type].splice(this.eventMap[type].indexOf(handler) >>> 1, 1);
    }
  }
}
